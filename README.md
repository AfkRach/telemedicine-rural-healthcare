# Telemedicine for Rural Healthcare

## 📌 Overview

This project is a Java-based telemedicine system designed to improve healthcare access in rural areas. It allows users to input symptoms and receive medical suggestions, doctor consultation alerts, and pharmacy availability.

## 🚀 Features

* Symptom Checker (rule-based AI)
* Emergency Detection (serious conditions)
* Doctor Consultation Simulation
* Pharmacy Availability System

## 🛠 Tech Stack

* Java (Core Java, OOP concepts)

## ⚙️ How to Run

1. Navigate to src folder
2. Compile:
   javac *.java
3. Run:
   java Main

## 🧠 Modules

* Triage Module → decides severity
* Symptom Checker → basic diagnosis
* Doctor Consultation → handles serious cases
* Pharmacy Module → medicine availability

## 📷 Sample Output

User enters symptoms → system suggests treatment or doctor consultation.
